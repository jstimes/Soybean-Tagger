Image Tagger Documentation & Instructions

About

Soybean Tagger is an application used for marking images with several different disease options. It cycles through a directory of leaf images, allowing users to mark on them where they believe certain infected areas are.

This site has been tested in Chrome & Firefox on both Windows and Mac. The libraries used do not support IE. If you experience issues on another platform, reach out to me at the email address below with some information about the problem.

This was developed by Jacob Stimes. If you experience any issues, have questions, or would like to share feedback, please reach out to me at jacob.stimes@gmail.com

General Instructions:

  1. Drag the diseases you identify into the 'Identified diseases' list. The currently selected disease has a red border around it.
  2. Click and drag on the image where you believe the selected disease is located. Upon releasing the mouse, the path will be closed. Upon selecting a disease, an input box will appear on screen where a percentage can be entered to reflect disease severity.
  3. Add more diseases if desired by dragging another option onto the identified diseases. Continue marking diseases you believe are in the image.
  4. Once you are done marking an image, click 'Save/Next Image'. The marked image will be replaced with the next one, and all data is reset. Continue this procedure to keep marking images, your progress is shown in the upper right corner.
  5. A message will be shown when you have marked all images.

Options and Controls:

  • 'Show Selected Disease Only' - If checked, will only show markings on the image for the currently selected disease (all markings will still be drawn on the output image. If you'd like an image for each disease, send me an email to the above address and I could add that feature). Otherwise all markings are shown.
  • 'Fill selection' - If checked, marked paths are filled in.
  • 'Clear' - resets all markings and data for the current image
  • 'Clear Selection' - resets only the markings and data for the currently selected disease (useful if you've already made several marks, but made one mistake).
  • 'Undo' - removes the last drawn marking, and data associated with it
